# Hangman_game

The program is a simple, console game and the variation of the Hangman game.

1. The random book's title is being selected from the text file.
2. The book's author is read from another text file.
3. Player guesses the book title by choosing letters (similar to the Hangman game).
4. After guessing the title correctly, the player is asked to deliver the name of the book's author.

Additional notes:
1. Player can choose how many times they will be allowed to guess the letter.
